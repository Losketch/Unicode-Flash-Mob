如果你安装了python Windows电脑请按Win+R，输入cmd，然后安装以下库
以下是运行前所需安装Python库的代码：
pip install requests
pip install fonttools
pip install Pillow
pip install tqdm
pip install ffmpeg-python
pip install ffmpeg
请确保在运行代码之前已经安装了Python和pip，并且已经配置好了正确的环境。
如果网络不好使用下面的安装代码
pip3 install requests -i https://pypi.tuna.tsinghua.edu.cn/simple/
pip3 install fonttools -i https://pypi.tuna.tsinghua.edu.cn/simple/
pip3 install Pillow -i https://pypi.tuna.tsinghua.edu.cn/simple/
pip3 install tqdm -i https://pypi.tuna.tsinghua.edu.cn/simple/
pip3 install ffmpeg-python -i https://pypi.tuna.tsinghua.edu.cn/simple/
pip3 install ffmpeg -i https://pypi.tuna.tsinghua.edu.cn/simple/

目录里的exe文件只是体验功能用的，真正想要生成视频还是要安装python并且已经配置好了正确的环境和安装了上面的第三方库
